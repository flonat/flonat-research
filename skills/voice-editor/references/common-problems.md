# Common Voice-Editing Problems

## Too Formal
**Symptom:** Reads like a corporate memo
**Fix:** Add contractions, shorten sentences, replace "individuals" with "people", "utilize" with "use"

## Too Uniform
**Symptom:** Every sentence same length, monotonous rhythm
**Fix:** Vary deliberately — 5-word punch, then 25-word exploration. Add one-sentence paragraphs for emphasis.

## Too Hedged
**Symptom:** Everything qualified, no strong positions
**Fix:** Delete "it might be argued", "generally speaking". Make direct claims. Replace "might" with "will" where confident.

## Too Balanced
**Symptom:** Every section same length, all points equal weight
**Fix:** Expand important points, shrink minor ones. Lead with strongest material. Let structure reflect priorities.

## Voice Drift
**Symptom:** Starts strong, becomes generic by end
**Fix:** Edit in 500-word chunks. Re-inject voice markers every few paragraphs. Check last third especially carefully.
